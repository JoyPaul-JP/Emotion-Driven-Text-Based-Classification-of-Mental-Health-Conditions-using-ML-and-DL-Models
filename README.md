The project titled "Emotion-Driven Text-Based Classification of Mental Health Conditions using ML and DL Models" focuses on building a reliable and efficient system to identify mental health conditions from social media text using emotion-based analysis. By using lexicons such as AFINN, VADER, and NRC, the system captures different emotional patterns, which helps in accurately distinguishing closely related classes like depression and suicide. The approach evaluates eight machine learning and eight deep learning models, achieving up to 92% accuracy in multi-class classification, with Artificial Neural Networks (ANN) providing the most balanced and consistent results. The project is developed in Python using Jupyter Notebook and uses tools such as Pandas, NumPy, Scikit-learn, Seaborn, and Matplotlib for data processing, model training, and result visualization. It is designed to support early detection of mental health conditions beyond traditional clinical methods, providing a practical and scalable solution for real-world applications.
